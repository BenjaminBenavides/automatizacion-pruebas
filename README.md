# Automatización de pruebas 

# Proyecto de ejemplo que implementa una calculadora simple con pruebas unitarias automatizadas usnado Junit y Maven 

# Tecnologias usadas
- Java
- Maven
- Junit
- Github Actions

# Se implementaron pruebas automatizadas para validar: 
- Suma de numeros con dos parametros
- Resta de numeros con dos parametros

# Integracion continua 
Se configuró el pipeline en github actions que ejecuta las pruebas automaticamente en cada push

# Para ejecutar el proyecto en la consola
mvn test
